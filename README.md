# 📊 HR Analysis Dashboard | Power BI

## 🚀 Overview

The HR Analysis Dashboard is an interactive Power BI project developed to analyze workforce data and provide actionable HR insights. The dashboard enables users to explore employee information through dynamic visualizations, filters, and KPIs.

🎯 Goal: Transform raw HR data into meaningful business insights for data-driven decision-making.

---

## 🏠 Dashboard Navigation Page

The first page acts as a dashboard landing page and includes:

✅ Navigation button to HR Analysis Dashboard  
✅ Data Source Information  
✅ Dashboard Refresh Frequency  
✅ Last Refresh Date  
✅ Contact Information  
✅ Project Overview

---

## 📈 HR Analysis Dashboard

The main dashboard provides detailed workforce analytics with interactive filtering capabilities.

### 🎛️ Filters

Users can filter data by:

- 📅 Date
- 👤 Employee Name
- 🏢 Business Unit
- 🌍 Country

---

## 📌 Key KPI

### 👥 Total Employees
Displays the total number of employees available in the dataset.

---

## 📊 Visualizations

### 📈 Employee Headcount Volume Trend
Tracks employee growth trends over time.

### 👨‍💼👩‍💼 Employee Headcount by Gender
Displays workforce gender distribution using a donut chart.

### 🎂 Employee Headcount by Age
Shows employee distribution across different age groups.

### 🌎 Employee Headcount by Country
Visualizes employee count across countries.

### 🏢 Employee Headcount by Business Unit
Provides workforce allocation across business units using a treemap.

### 📋 Employee Headcount by Department
Displays employee distribution across organizational departments.
